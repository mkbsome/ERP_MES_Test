# MES/ERP 시스템 모듈 분석 보고서

## 1. 참조 시스템 분석 (6개 스크린샷 기반)

### 1.1 Live-MES (스크린샷 1-2)

**기초정보관리**
- 품목정보, 거래처정보, 창고/위치정보
- BOM관리, 공정정보, 설비정보
- 작업자정보, 불량유형, 검사기준

**생산관리**
- 작업지시, 생산실적, 공정진행현황
- 작업일보, LOT추적

**영업관리**
- 수주등록, 수주현황, 출하관리
- 납기관리

**품질관리**
- 수입검사, 공정검사, 출하검사
- 불량등록, 시정조치(CAR)

**자재관리**
- 재고현황, 입출고관리
- 재고조정, 안전재고

**외주관리(SCM)**
- 외주발주, 외주입고
- 외주검사, 외주비관리

**설비관리**
- 설비마스터, 설비현황
- 설비점검, 보전이력
- 가동률/OEE

**구매관리**
- 발주등록, 발주현황
- 입고등록, 매입관리

**KPI관리**
- 생산KPI, 품질KPI
- 설비KPI, 대시보드

**POP (Point of Production)**
- 작업시작/종료, 실적등록
- 불량등록, 자재투입

---

### 1.2 Live-POP (스크린샷 3)

**자재관리**
- 자재입고, 자재출고
- 재고현황, 자재요청

**외주관리**
- 외주발주, 외주입고
- 외주검사

**생산관리**

- 생산실적, 공정이동
- LOT분할/병합

**현황관리**
- 라인현황, 설비현황
- 생산현황, 재고현황

**영업관리**
- 출하지시, 출하등록
- 납품현황

---

### 1.3 Live-LMS (스크린샷 4) - 설비연동시스템

**기준정보**
- 설비등록, 신호정의
- 수집항목, 알람설정

**설비신호**
- 신호수집, 신호모니터링
- 가동상태, 알람이력

**모니터링**
- 실시간현황, 가동현황
- 설비상태, 알람현황

---

### 1.4 Live-EDGE (스크린샷 5) - 데이터수집시스템

**프로젝트관리**
- 프로젝트등록, 설정관리
- 연결관리

**데이터수집**
- 수집설정, 수집현황
- 데이터모니터링

**기준정보관리**
- 장비등록, 태그정의
- 수집규칙

**운영정보**
- 수집이력, 오류이력
- 통계정보

### 1.5 PMS 모듈 (스크린샷 6) - ERP 시스템

**운영관리**
- 코드관리, 사용자관리

**영업관리**
- 견적관리, 수주관리

**회계관리**
- 전표관리, 계정관리

**구매/물류**
- 발주관리, 입고관리
- 재고관리, 출고관리

**인사/급여**
- 인사정보, 근태관리
- 급여계산, 4대보험

**프로젝트관리**
- 프로젝트등록, 일정관리
- 원가관리, 진행현황

## 3. 비교 분석

### 3.1 ERP 모듈 비교

| 사용자 제안 | 참조 시스템 | 차이점/권고 |
|------------|------------|------------|
| 재무/회계 | PMS: 회계관리 | ✅ 일치 |
| 구매관리 | Live-MES: 구매관리 | ✅ 일치 |
| 영업/판매관리 | PMS: 영업관리 | ✅ 일치 |
| 재고관리 | PMS: 구매/물류 | ✅ 일치 (물류에 포함) |
| 생산관리 | - | ⚠️ ERP 생산관리 = MPS/MRP/작업지시 발행까지 |
| 기준정보관리 | PMS: 운영관리 | ✅ 일치 (코드/사용자/권한) |
| 인사/급여 | PMS: 인사/급여 | ✅ 일치 |

### 3.2 MES 모듈 비교

| 사용자 제안 | 참조 시스템 | 차이점/권고 |
|------------|------------|------------|
| 작업지시/생산실행 | Live-MES: 생산관리 | ✅ 일치 |
| 생산실적관리 | Live-POP: 생산관리 | ✅ 일치 |
| 불량관리 | Live-MES: 품질관리 일부 | ✅ 일치 |
| LOT/이력추적 | Live-MES: 생산관리 일부 | ✅ 일치 |
| 공정관리 | Live-MES: 기초정보 일부 | ✅ 일치 |
| 품질검사관리 | Live-MES: 품질관리 | ✅ 일치 |
| 작업자관리 | Live-MES: 기초정보 일부 | ✅ 일치 |
| 설비상태 | Live-LMS + Live-MES: 설비관리 | ✅ 일치 |

### 3.3 참조 시스템에 있으나 사용자 제안에 없는 기능

**MES 측면**
- 외주관리(SCM) - 외주가공 업체와의 연동
- KPI관리/대시보드 - 성과지표 모니터링
- POP 전용 화면 - 현장 터치스크린용

**ERP 측면**
- 프로젝트관리 - 개발/수주 프로젝트 (제조업에서는 선택적)

## 5. PCB/SMT 제조 특화 설계

### 5.1 공정 흐름 (그린보드 일렉트로닉스)

```
┌─────────────────────────────────────────────────────────────────┐
│                    SMT (Surface Mount Technology) 라인           │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│  [PCB투입] → [Loader] → [Printer] → [SPI] → [Mounter]          │
│                            ↓          ↓         ↓               │
│                       솔더 도포   검사1    부품실장              │
│                                                                  │
│  → [Reflow] → [AOI] → [Unloader] → [검사/수리] → [완성]        │
│        ↓        ↓                                               │
│     솔더링   검사2                                              │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

### 5.2 주요 설비 목록

| 설비코드 | 설비명 | 용도 | 수집 데이터 |
|----------|--------|------|-------------|
| SMT-LOAD-01 | PCB Loader | PCB 공급 | 투입수량, 에러 |
| SMT-PRNT-01 | Solder Printer | 솔더 페이스트 도포 | 인쇄압력, 속도, 위치 |
| SMT-SPI-01 | SPI | 솔더 검사 | 볼륨, 높이, 면적, 불량 |
| SMT-CHIP-01 | Chip Mounter | 소형 부품 실장 | 실장수, 흡착에러, 인식에러 |
| SMT-CHIP-02 | Multi Mounter | 대형/이형 부품 | 실장수, 에러 |
| SMT-REFL-01 | Reflow Oven | 솔더링 | 온도프로파일, 컨베이어속도 |
| SMT-AOI-01 | AOI | 외관 검사 | 불량유형, 불량위치 |
| SMT-UNLD-01 | Unloader | PCB 취출 | 산출수량 |

### 5.3 PCB 특화 불량 유형

| 코드 | 불량유형 | 원인 | 관련설비 |
|------|----------|------|----------|
| DF01 | Bridge (솔더브릿지) | 솔더 과다 | Printer, Reflow |
| DF02 | Open (미납) | 솔더 부족 | Printer |
| DF03 | Tombstone (직립) | 열 불균형 | Reflow |
| DF04 | Missing (부품 누락) | 흡착 실패 | Mounter |
| DF05 | Shift (위치 이탈) | 실장 오차 | Mounter |
| DF06 | Wrong Part (오삽) | 피더 오류 | Mounter |
| DF07 | Polarity (극성 반대) | 인식 오류 | Mounter |
| DF08 | Cold Solder (냉납) | 온도 부족 | Reflow |
| DF09 | Crack (크랙) | 열 스트레스 | Reflow |
| DF10 | Void (보이드) | 가스 발생 | Reflow |

## 6. AI 플랫폼 연계 포인트 (참고)

> 참고: AI 플랫폼은 별도 개발이며, ERP/MES는 데이터 제공 역할

### 6.1 MES → AI 플랫폼 데이터

| 데이터 | 용도 | 트리거 조건 |
|--------|------|-------------|
| 불량 데이터 | 품질 이상 판단 | 불량률 > 2.5% |
| 설비 OEE | 설비 건강 판단 | OEE < 75% |
| 생산 실적 | 생산성 분석 | 목표 대비 미달 |
| 공정 데이터 | 공정 최적화 | 주기적 분석 |

### 6.2 ERP → AI 플랫폼 데이터

| 데이터 | 용도 | 트리거 조건 |
|--------|------|-------------|
| 재고 현황 | 수요 예측 | 안전재고 도달 |
| 수주 정보 | 납기 분석 | 긴급 수주 |
| 원가 데이터 | 원가 분석 | 목표 대비 초과 |

## 7. 결론

### 7.1 사용자 제안 모듈 평가
- ERP 7개 모듈: **적절함** (표준 ERP 구조)
- MES 8개 모듈: **적절함** (MES 핵심 기능 포함)

### 7.2 권고 사항
1. **외주관리**: 필요시 MES에 추가 (SCM 연동)
2. **KPI/대시보드**: 공통 모듈로 추가 권장
3. **POP 화면**: MES의 "작업지시/생산실행"에 포함
4. **설비연동**: MES "설비상태" 모듈에서 처리

### 7.3 최종 화면 수
- **ERP**: 7개 모듈, 28개 화면
- **MES**: 8개 모듈, 32개 화면
- **총계**: 15개 모듈, 60개 화면
