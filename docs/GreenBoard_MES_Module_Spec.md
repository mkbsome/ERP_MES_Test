# 그린보드 일렉트로닉스 MES 모듈 상세 설계서

## 1. 시스템 개요

| 항목 | 내용 |
|------|------|
| **시스템명** | GreenBoard MES (Manufacturing Execution System) |
| **대상 기업** | (주)그린보드 일렉트로닉스 |
| **업종** | PCB/전자부품 제조 (SMT 라인) |
| **규모** | 2개 공장, 15개 생산라인 (SMT 10, THT 2, 조립 3) |

---

## 2. MES 모듈 구성도

```
┌─────────────────────────────────────────────────────────────────────────────────────┐
│                                      MES                                             │
├──────────┬──────────┬──────────┬──────────┬──────────┬──────────┬──────────────────┤
│기준정보  │ 생산관리  │ 품질관리  │ 설비관리  │ 현장관리  │  I/F     │ 시스템관리      │
│관리      │          │          │          │ (POP)    │ (연계)   │                  │
├──────────┼──────────┼──────────┼──────────┼──────────┼──────────┼──────────────────┤
│ 12개     │  14개    │  12개    │  10개    │  12개    │  6개     │  8개             │
│ 화면     │  화면    │  화면    │  화면    │  화면    │  화면    │  화면            │
└──────────┴──────────┴──────────┴──────────┴──────────┴──────────┴──────────────────┘
                                    총 74개 화면
```

## 4. PCB/SMT 특화 기능

### 4.1 SMT 라인 공정 흐름

```
┌──────────────────────────────────────────────────────────────────────────────┐
│                           SMT 생산 라인 흐름                                  │
├──────────────────────────────────────────────────────────────────────────────┤
│                                                                              │
│  [PCB투입]    [Loader]    [Printer]    [SPI]      [Mounter]                 │
│     │            │            │          │            │                      │
│     ▼            ▼            ▼          ▼            ▼                      │
│   원자재      PCB공급     솔더도포    솔더검사    부품실장                    │
│   LOT투입                  (Paste)    (Paste     (Chip/                      │
│                                      Inspection) Multi)                      │
│                                                                              │
│  [Reflow]    [AOI]     [Unloader]   [수리]      [출하검사]    [완성]        │
│     │          │           │          │            │            │           │
│     ▼          ▼           ▼          ▼            ▼            ▼           │
│   솔더링     외관검사    PCB취출    불량수리     최종검사     완제품         │
│  (열처리)   (Optical)                                        LOT생성        │
│                                                                              │
└──────────────────────────────────────────────────────────────────────────────┘
```

### 4.2 주요 설비 목록

| 설비코드 | 설비명 | 용도 | 수집 항목 |
|----------|--------|------|-----------|
| SMT-LOAD-01~02 | PCB Loader | PCB 공급 | 투입수, 에러횟수 |
| SMT-PRNT-01~02 | Screen Printer | 솔더 페이스트 도포 | 인쇄압력, 속도, 클리닝주기 |
| SMT-SPI-01~02 | SPI | 솔더 검사 | 볼륨, 높이, 면적, 불량수 |
| SMT-CHIP-01~04 | Chip Mounter | 소형부품 실장 | 실장수, 흡착에러, 인식에러, 피더상태 |
| SMT-MULT-01~02 | Multi Mounter | 이형부품 실장 | 실장수, 에러율 |
| SMT-REFL-01~02 | Reflow Oven | 솔더링 | 온도프로파일(8존), 컨베이어속도 |
| SMT-AOI-01~02 | AOI | 외관 검사 | 검사수, 불량유형별 수, False Call율 |
| SMT-UNLD-01~02 | Unloader | PCB 취출 | 산출수 |

## 5. AI 플랫폼 연계 포인트

### 5.1 데이터 제공 (MES → AI)

| 데이터 | 연계 화면 | 트리거 조건 | 용도 |
|--------|----------|------------|------|

| 설비 OEE | MES-EQP-006 | OEE < 75% | 설비 건강 판단 |
| SPI/AOI 검사 | MES-QUA-001~002 | 실시간 | 품질 예측 |

## 7. 개발 우선순위

### Phase 1: 핵심 기능 (30개 화면)
- 기준정보: 코드, 공정, 품목, 작업자, 불량유형 (6개)
- 생산관리: 생산지시, 작업지시, 실적등록/조회 (6개)
- 품질관리: 불량등록/조회, SPI/AOI 결과 (6개)
- 설비관리: 설비정보, 현황모니터링, OEE (4개)
- 현장관리: 작업시작/종료, 실적등록 (4개)
- 시스템: 사용자, 권한 (4개)

### Phase 2: 확장 기능 (24개 화면)
- 생산관리: 계획, 가동현황, LOT추적 (6개)
- 품질관리: SPC, 자주검사, 부적합품 (4개)
- 설비관리: 보전관리, 고장내역 (4개)
- 현장관리: 나머지 POP 화면 (6개)
- I/F: ERP 연계 (4개)

### Phase 3: 고도화 (20개 화면)
- 나머지 전체 화면
- AI 연계 강화
- 리포트/분석 기능
